Carcassonne Player Guide
The Game

Carcassonne is a popular German game created by Klaus-Jürgen Wrede and published by Hans im Glück Verlag.

This module includes Carcassonne (base game), The River, Inns & Cathedrals, Traders & Builders, The King, The Cathars, The Count of Carcassonne, The Princess and The Dragon, The River II, The Tower, and the GQ Mini expansion.

Usage of this module requires that you own a hard copy of the game.

For updates, support, or to find players visit http://carcassonne.locehilios.com/.

The Interface

You will find the following screens within the module:

The Message Window
The message window is where all messages concerning the game are printed, and allows you to chat with your opponents. You can not hide the message window.
The Setup Window
The setup window description. You can toggle the setup window display by clicking on the "Setup" button or by pressing CTRL-S.
The Tile Window
The tiles window is where you draw tiles from during the game. There's a "Draw" stack (for selecting a new tile), and a "Stage" stack (to hold newly drawn tiles). You can toggle the tiles window display by clicking on the "Tiles" button or by pressing CTRL-T. The number of tiles remaining is displayed below a stack.
The River Tiles Window
The river tiles window is where you draw river tiles from during the game. There's a "Draw" stack (for selecting a new river tile), and a "Stage" stack (to hold newly drawn tiles). There are also "Lake" and "Spring" stacks which contain those special river tiles. If you want to draw a specific Lake or Spring tile, right click on the stack and select "Draw specific cards", select the tile you want, and then draw from the stack. You can toggle the river tiles window display by clicking on the "River Tiles" button or by pressing CTRL-V. The number of tiles remaining is displayed below a stack.
The Game Window
The game window contains the scoring track, player supplies, and extra pieces that are needed for some expansions. You can toggle the game window display by clicking on the "Game Window" button or by pressing CTRL-G. Most tokens are grouped into stacks. The number of tokens is displayed below a stack.
The Table
This is the area in which tiles are placed into play. There is a visible grid on the table which shows you where tiles will be placed. A square is shown in the center of the table (to indicate where to place your first tile). The "zoom" of the table can be modified by clicking on the zoom buttons (magnifying glass). In addition, you can view the entire table with the table overview (which can be helpful during large games). You can toggle the table overview display by clicking on the "Overview" button or by pressing CTRL-SHIFT-O. You can not hide the table. The standard start tile is automatically placed in the center of the table. NOTE: Some expansions cause the standard start tile to be removed.
Game Setup

To set up a new game, choose the "New Game" item from the "File" menu. After a game has been created, expansions can be added. Click on the expansion you wish to add. The needed tiles and tokens will be added to the game. If desired, tokens for colors not used can be removed. Click on the meeple of the color you wish to remove.

Game Play

Once a game has been created and players have connected (see the vassal help for how to play online) the game can begin.

A player generally begins their move by dragging a tile from "Draw" stack to the "Stage" stack on the "Tiles" window. (At the beginning of a game tiles might be drawn instead from the "River Tiles" window")

Placing the tile in the "Stage" stack allows opponents to view the tile as the player contemplates how they will place the tile. After contemplation, the player drags the tile from the "Stage" stack to the Table. On the table, tiles will automatically snap to the visible grid. In the "stage" stack or the table, tiles can be rotated via the right-click menu or by pressing CTRL-[ and CTRL-].

Once a player is satisfied with the position and orientation of their tile, they often are allowed to place a token onto that tile. To do so, drag the token from the game window to the table. Player's meeples (followers) have two positions; standing and lying down. Generally, players indicate a meeple is used as a farmer by changing the position of the meeple. The position of meeples can be changed via the right-click menu or by pressing CTRL-L.

When a player's turn is complete, finish the move by pressning the "Finished" button or by pressing CTRL-F. Finishing in this way will "place" the tiles, and mark the last moved tile with your color. This prevents the tiles from being moved with in the future. If you "placed" a tile or "finished" by mistake, click the Undo button.

Scoring

Scoring is done on the Game Window. Pressing CTRL-= will print the current scores in the message window. There are two options for moving your score meeple around the scoring track.

Hot Key Scoring
Pressing CTRL and any number will advance YOUR score meeple that many spaces (0 will advance your meeple 10 spaces). If you need to score yourself 8 points, press CTRL-8. 10 points, press CTRL-0. 14 points, press CTRL-7 twice. You do not need to have the game window open to score this way, and the scores will be reported in the message window. You can only control your OWN score meeple in this way.
Manual Scoring
Just drag the scoring meeple to another spot on the score track. Whenever scoring meeples share the same spot on the scoring track, they "stack together". To select individual scoring meeples when stacked, first double-click on the stack of scoring meeples, then select your meeple to move it. As you move your scoring meeples on the score track, the game will report scoring changes in the message window. If you complete one trip around the score track you can increase your score by 50 points via the right-click menu or by pressing CTRL-].
Shifting the Table

In general, if you start in the center of the table there will be plenty of room to play the game. However, if you run out of room, you can shift the entire table in any direction. (This is optional, and some players like to play bound by the table size. Players should agree prior to the game if they will allow the table to be shifted.) Be sure to use the "Overview" feature while you shift the table so you don't shift some pieces off the table! If you do, you can use Undo to fix that. Shift the table by pressing the buttons with the thick black arrows or by pressing CTRL plus any of the arrow keyboard keys.
