Carcassonne Player Guide
The Game

Carcassonne is a popular German game created by Klaus-Jürgen Wrede and published by Hans im Glück Verlag.

This module includes Carcassonne (base game), The River, Inns & Cathedrals, Traders & Builders, The King, The Cathars, The Count of Carcassonne, The Princess and The Dragon, The River II, The Tower, and the GQ Mini expansion.

Usage of this module requires that you own a hard copy of the game.

For updates, support, or to find players visit http://carcassonne.locehilios.com/.

The Interface
You will find the following screens within the module:
The Message Window
The message window is where all messages concerning the game are printed, and allows you to chat with your opponents. You can not hide the message window.
The Setup Window
The setup window description. You can toggle the setup window display by clicking on the "Setup" button or by pressing CTRL-S.
The Tile Window
The tiles window is where you draw tiles from during the game. There's a "Draw" stack (for selecting a new tile), and a "Stage" stack (to hold newly drawn tiles). You can toggle the tiles window display by clicking on the "Tiles" button or by pressing CTRL-T.
The River Tiles Window
The river tiles window is where you draw river tiles from during the game. There's a "Draw" stack (for selecting a new river tile), and a "Stage" stack (to hold newly drawn tiles). There are also "Lake" and "Spring" stacks which contain those special river tiles. If you want to draw a specific Lake or Spring tile, right click on the stack and select "Draw specific cards", select the tile you want, and then draw from the stack. You can toggle the river tiles window display by clicking on the "River Tiles" button or by pressing CTRL-V.

The Game Window
The game window contains the scoring track, player supplies, and extra pieces that are needed for some expansions. You can toggle the game window display by clicking on the "Game Window" button or by pressing CTRL-G.

The Table
This is the area in which tiles are placed into play. There is a visible grid on the table which shows you where tiles will be placed. A square is shown in the center of the table (to indicate where to place your first tile). The "zoom" of the table can be modified by clicking on the zoom buttons. In addition the "Overview" button will toggle the display of the table overview (which can be helpful during large games). You can not hide the table.

The standard start tile is automatically placed in the center of the table. NOTE: Some expansions cause the standard start tile to be removed.

Setup
To set up a new game, choose the "New Game" item from the "File" menu. Scenario packs add menu items, each a predefined game scenario ready to use. These predefined scenarios can be used as they are, or you can modify them after loading one. There is also a "Custom" choice which creates a new game with all players available, and all pieces in the game box.

Adding/Removing Tiles
After a game has been created, tiles can be added (moved to the tile stacks / table) or removed (returned to the game box). Entire stacks of tiles can be added to the game by right-clicking on a stack, and choosing "Add to game". Tiles can also be added to the game by dragging them from the game box, to the tile stacks window. For certain game scenarios, it is desirable to reserve particular tiles for specific phases of the game, for example the start of the game. Generally, these special tiles appear at the top of the game box stacks. Specific tiles can also be drawn from game box stacks by right-clicking on a stack and choosing "Draw specific cards". Special use tiles are generally placed in the "Stage" stack to begin. Other start tiles, such as non-termination river tiles, are placed in the "Start" stack. All other tiles for the game are placed in the "Draw" stack.

Entire sets of tiles can be removed from the game by clicking the "Tools" button, and selecting a set of tiles from the "Tile Removal" menu.
NOTE: Tiles in the game window or tiles which have been "Placed" can not be removed. Tiles should only be removed in this way prior to the start of a game.

Adding/Removing Pieces
After a game has been created, pieces can be added (moved to the game window) or removed (returned to the game box). Single pieces can be added to the game by selecting them in the game box, and choosing the appropriate menu item. In general CTRL-A, is the HOTKEY used to add a piece to the game. You can also select multiple pieces and press CTRL-A to add all the pieces.

Pieces can be removed by selecting one or more of them and choosing "Remove from game" or pressing CTRL-SHIFT-R.

Game Play
Once a game has been created and players have connected (see the vassal help for how to play online) the game can begin.

A player generally begins their move by dragging a tile from "Draw" stack to the "Stage" stack. (At the beginning of a game this might be skipped because some pieces may already exist in the "Stage" stack) Placing the tile in the "Stage" stack allows opponents to view the tile as the player contemplates how they will place the tile. After contemplation, the player drags the tile from the "Stage" stack to the Table. On the table, tiles will automatically snap to the visible grid. Once on the table, tiles can be rotated via the right-click menu or by pressing CTRL-[ and CTRL-]. Once a player is satisfied with the position and orientation of their tile, they "place" the tile via the right-click menu or by pressing CTRL-P. Once placed, the tile will have a thick red outline. This outline always surrounds the most recently "placed" tile. Once a tile is "placed" it can not be moved or rotated. If a tile is "placed" in error, click on "Undo" twice, reposition it, and then "place" it again.

Often after placing a tile, a player is allowed to place a piece onto that tile. To do so, drag the piece from the game window to the table. Player's meeples (followers) have two positions; standing and lying down . Generally, players indicate a meeple is used as a farmer by changing the position of the meeple to it is clear the meeple is in the field, and not on some other feature. Change a meeples position via the right-click menu or by pressing CTRL-L.

When features are scored, pieces can be returned to their owner's supplies via the right-click menu or by pressing CTRL-A.

When a player has finished their move, the easiest way to signal that is pressing the "Done" button or by pressing CTRL-D.

Some pieces from expansion packs have special features. Please read the expansion-specific guide for information about those pieces.

Scoring
Whenever scoring meeples share the same spot on the scoring track, they "stack together". To select individual scoring meeples when stacked, first double-click on the stack of scoring meeples, then select your meeple to move it. As you move your scoring meeples on the score track, the game will report scoring changes in the message window. If you complete one trip around the score track you can increase your score by 50 points via the right-click menu or by pressing CTRL-].
Shifting the Table
In general, if you start in the center of the table there will be plenty of room to play the game. However, if you run out of room, you can shift the entire table in any direction. (This is optional, and some players like to play bound by the table size. Players should agree prior to the game if they will allow the table to be shifted.) Be sure to use the "Overview" feature while you shift the table so you don't shift some pieces off the table!

Shift the table by using the arrow buttons , or by pressing CTRL and any of the arrow keyboard keys.
